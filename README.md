# Birthday Greeting Web App

Aplikasi web ini dirancang untuk mengirimkan ucapan selamat ulang tahun kepada teman. Aplikasi ini memiliki animasi dan pengalaman interaktif untuk membuat pesan ulang tahun menjadi istimewa.

## Struktur Proyek

```
birthday-greeting-app
├── src
│   ├── index.html        # Dokumen HTML utama untuk aplikasi
│   ├── styles.css        # Gaya untuk aplikasi web
│   ├── script.js         # Logika JavaScript untuk aplikasi
│   └── assets
│       └── animations    # Direktori untuk file-file animasi
├── package.json          # File konfigurasi npm
└── README.md             # Dokumentasi proyek
```

## Memulai

Untuk memulai dengan Aplikasi Ucapan Selamat Ulang Tahun, ikuti langkah-langkah berikut:

1. **Klon repositori**:
   ```
   git clone <repository-url>
   cd birthday-greeting-app
   ```

2. **Instal dependensi**:
   Jika Anda memiliki npm terinstal, jalankan perintah berikut untuk menginstal dependensi yang diperlukan:
   ```
   npm install
   ```

3. **Buka aplikasi**:
   Buka `src/index.html` di browser web Anda untuk melihat ucapan selamat ulang tahun.

## Fitur

- Ucapan selamat ulang tahun yang dirancang dengan indah
- Animasi yang menarik untuk meningkatkan pengalaman pengguna
- Desain yang sederhana dan interaktif
